# Octave Engineering Mathematics

This repository contains the code and documentation for the **GNU Octave Engineering Mathematics Laboratory Sessions**. 

The sessions focus on implementing mathematical concepts computationally to solve core engineering problems using GNU Octave.

---

### Prerequisites
To run these scripts, you need to have **GNU Octave** installed on your system. 
* Download it from the [Official GNU Octave Website](https://octave.org/).
* Alternatively, install it via your terminal:
  ```bash
  # For Ubuntu/Debian
  sudo apt install octave

  # For macOS (using Homebrew)
  brew install octave
