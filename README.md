## 📖 Overview

This repository contains the code and documentation for the **GNU Octave Engineering Mathematics Laboratory Sessions**. 

The sessions focus on implementing mathematical concepts computationally to solve core engineering problems using GNU Octave. It serves as a structured repository for lab exercises, matrix computations, numerical methods, and data visualizations.

---


##  Prerequisites

To run these scripts, you need to have **GNU Octave** installed on your system.

### Installation

*   **Official Installer:** Download it directly from the [Official GNU Octave Website](https://www.gnu.org/software/octave/).
*   **Terminal (Ubuntu/Debian):**
    ```bash
    sudo apt update
    sudo apt install octave
    ```
*   **Terminal (macOS using Homebrew):**
    ```bash
    brew install octave
    ```
